자료형과  유사하게 선언하는것을 SCALAR 라고함

```
DECLARE
	VEMPNO NUMBER(4);
	VENAME VARCHAR2(10);
BEGIN
	VEMPNO := 7788;
	VENAME := 'SCOTT';
	DBMS_OUTPUT.PUT_LINE('사번/이름');
	DBMS_OUTPUT.PUT_LINE('-----------');
	DBMS_OUTPUT.PUT_LINE(VEMPNO || '/' || VENAME);
END;
/
```


## 레퍼런스

```
VEMPNO EMP.EMPNO%TYPE; /  VEMPNO 에 EMPNO 타입과 맞는 자료형 선언

VEMPNO EMP%ROWTYPE;   / EMP 로우를 참조
```
