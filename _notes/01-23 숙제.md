## 탄탄히 다지기 3번 부서 번호를 전달하여 해당 부서 소속 사원의 정보를 출력하는 SEL_EMP 프로시저 커서를 사용하여 작성해 보세요.

```
CREATE OR REPLACE PROCEDURE SEL_EMP( VDEPTNO IN EMP.DEPTNO%TYPE)


IS
	VEMP EMP%ROWTYPE;
	CURSOR C1
	IS
	SELECT EMPNO,ENAME,JOB,SAL FROM EMP
	WHERE VDEPTNO = DEPTNO;
BEGIN
	DBMS_OUTPUT.PUT_LINE('사번 / 사원명 / 직급 /   급여 ' );
	DBMS_OUTPUT.PUT_LINE('----------------------------- ' );

	OPEN C1;

	LOOP
		FETCH C1 INTO VEMP.EMPNO,VEMP.ENAME,VEMP.JOB,VEMP.SAL;
			
		
		EXIT WHEN C1%NOTFOUND;

		DBMS_OUTPUT.PUT_LINE(VEMP.EMPNO || ' / '||VEMP.ENAME || ' / '|| VEMP.JOB || ' / ' || VEMP.SAL );
	
	END LOOP;

	CLOSE C1;
END;
/
```
